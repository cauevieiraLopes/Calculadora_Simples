# Calculadora_Simples

## 📌 Descrição

Este projeto consiste em uma calculadora interativa em Python, que permite ao usuário realizar diferentes operações matemáticas diretamente pelo terminal.

O programa apresenta um menu com opções de operações e executa a escolhida pelo usuário. Após cada operação, o sistema retorna ao menu principal, permitindo novas operações até que o usuário decida encerrar o programa.

As operações disponíveis incluem:

Soma

Subtração

Divisão

Multiplicação

Potência

O programa é encerrado quando o usuário seleciona a opção de saída.

## ⚙️ Funcionalidades

Menu interativo de operações matemáticas

Entrada de dois valores numéricos

Execução de operações básicas:

adição

subtração

divisão

multiplicação

potência

Repetição contínua até o usuário escolher sair

Encerramento controlado do sistema

## 🧠 Lógica do Programa

O sistema utiliza um loop while controlado por uma condição booleana, permitindo que o programa continue executando até que o usuário escolha sair.

Fluxo básico:

O programa exibe um menu com as operações disponíveis.

O usuário escolhe uma opção.

O sistema verifica a operação escolhida:

Se for válida, solicita dois números.

Realiza o cálculo correspondente.

Exibe o resultado.

Se o usuário escolher a opção de saída, o programa é encerrado.

Caso uma opção inválida seja digitada, o sistema simplesmente reinicia o menu.
